# Explorador de Juegos Gratuitos

## Descripción

Esta es una galería de juegos gratuitos que te permite descubrir nuevos juegos. La aplicación consume la API de [FreeToGame](https://www.freetogame.com/) para mostrar una lista de juegos que se pueden jugar de forma gratuita. Al hacer clic en un juego, serás redirigido a su página web oficial para poder jugarlo.

## Tecnologías Utilizadas

*   **HTML**
*   **CSS**
*   **JavaScript**

## API

Este proyecto utiliza la [FreeToGame API](https://www.freetogame.com/api-doc) para obtener la información de los juegos.

## Cómo Ejecutar el Proyecto

1.  Clona o descarga este repositorio.
2.  Abre la carpeta del proyecto en tu editor de código (como Visual Studio Code).
3.  Si tienes la extensión "Live Server" en VS Code, haz clic derecho en el archivo `index.html` y selecciona "Open with Live Server".
